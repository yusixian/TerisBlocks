# TerisBlocks
c++综合案例_俄罗斯方块
能实现俄罗斯方块的移动、旋转、下落等基本功能。
有人机交互界面，游戏界面右侧有预览下个方块的区域。
有积分机制（一次性消除1行可得10积分，2行30积分，3行60积分，4行100积分）
还有一堆bug【划掉】
直接下载可执行文件就可以玩了✓
